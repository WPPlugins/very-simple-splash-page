== Description ==


Very Simple Splash Page is a simple plugin that lets you add a welcome page with background image or video to your existing theme. Its full responsive, very simple to install and use it. Add image or video with intro text and button. 


Features:

* Fully Responsive
* Add Background Image
* Add Background Video (mp4)
* Change font size and color
* Change button color
* Works with every WordPress Theme


== Installation ==

1. Upload very-simple-splash-page folder to wp-content/plugins.
2. Click "Activate" in the WordPress plugins menu.
3. Go to Very Simple Splash Page Options and check "Enable Splash Page" option.
4. Now you can customize your splash page.
